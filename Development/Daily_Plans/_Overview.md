# Daily Plans

**Purpose:** Day-by-day planning and progress tracking.

---

## How to Use This Folder

Create a new file for each planning session:

```
Daily_Plans/
├── _Overview.md          ← You are here
├── 2026-01-22.md         ← Today's plan
├── 2026-01-23.md         ← Tomorrow's plan
└── ...
```

---

## Daily Plan Template

Copy this template when creating a new daily plan:

```markdown
# Daily Plan — [DATE]

## Yesterday's Progress
- What got done
- What didn't get done
- Any blockers

## Today's Goals
- [ ] Goal 1
- [ ] Goal 2
- [ ] Goal 3

## Notes
- Any thoughts, decisions, or discoveries

## Tomorrow's Preview
- What's coming next
```

---

## Weekly Review Template

Create a weekly review every Sunday:

```markdown
# Week [NUMBER] Review — [DATE RANGE]

## Completed This Week
- Item 1
- Item 2

## Carried Over
- Item that didn't get done

## Blockers/Challenges
- Challenge faced and how it was resolved

## Next Week's Focus
- Main priority for the week
```
