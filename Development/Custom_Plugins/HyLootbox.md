# HyLootbox

Reward/lootbox system.

## Purpose
- Lootbox mechanics
- Reward distribution
- Rarity system

## Tiers
Common → Uncommon → Rare → Legendary

## Features
- Open lootboxes
- Animated reveal UI
- Duplicate protection (or conversion)
- Drop rate transparency

## Dependencies
- HyCore
- HyEconomy (Gems to purchase)
- HyInventory (rewards go here)
