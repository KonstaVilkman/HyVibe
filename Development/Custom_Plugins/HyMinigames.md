# HyMinigames

Minigame framework and games.

## Purpose
- Minigame infrastructure
- Queue/matchmaking
- Rewards integration

## Games (planned)
- Parkour
- Hide and Seek
- Racing
- Trivia
- Dance-off
- Building contests
- Others TBD

## Features
- Game queue system
- Lobby/waiting area
- Score tracking
- Leaderboards per game
- Rewards on completion

## Dependencies
- HyCore
- HyEconomy (rewards)
- HyFame (fame rewards)
