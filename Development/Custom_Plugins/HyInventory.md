# HyInventory

Custom inventory system for cosmetics and items.

## Purpose
- Separate cosmetic inventory from gameplay
- Custom UI for browsing owned items
- Equip/unequip cosmetics
- Hybrid with native inventory where needed

## Features
- Cosmetics tab (clothing, accessories)
- Furniture tab (for housing)
- Emotes tab
- Filters and search
- Preview before equip

## Dependencies
- HyCore
- HyCosmetics (integration)
