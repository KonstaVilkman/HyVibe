# HyEconomy

Dual currency + crystals system.

## Purpose
- Track player currencies
- Handle transactions
- Economy API for other plugins

## Currencies
- **Cash ($)** - Common, tradeable
- **Gems** - Premium, non-tradeable
- **Crystals** - Anchor currency, tradeable

## Features
- Balance checking
- Add/remove currency
- Transfer between players (tradeable only)
- Transaction logging

## Dependencies
- HyCore
