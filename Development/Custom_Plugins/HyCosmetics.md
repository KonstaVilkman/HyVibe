# HyCosmetics

Cosmetic application system.

## Purpose
- Apply cosmetics to player appearance
- Manage equipped cosmetics
- Handle cosmetic unlocks

## Categories
- Clothing
- Accessories
- Particles
- Name effects
- Titles
- Badges

## Features
- Equip/unequip
- Preview
- Unlock tracking
- Cosmetic slots

## Dependencies
- HyCore
- HyInventory (UI integration)
