# HyFame

Fame tracking and decay system.

## Purpose
- Track player fame points
- Calculate fame tiers
- Handle fame decay

## Tiers
Newcomer → Familiar → Known → Popular → Celebrated → Renowned → Iconic → Legend

## Features
- Earn fame from activities
- 2-day grace period before decay
- Tier calculation
- Fame leaderboard

## Dependencies
- HyCore
