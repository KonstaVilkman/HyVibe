# HyChat

Custom chat formatting.

## Purpose
- Display fame tier in chat
- Custom name colors
- Chat channels (future)

## Features
- Format: [Tier] [Title] PlayerName: message
- Title display
- Name color from cosmetics
- Mute/ignore (future)

## Dependencies
- HyCore
- HyFame (tier display)
- HyCosmetics (title/color)
- LuckPerms (moderation)
