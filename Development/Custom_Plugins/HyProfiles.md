# HyProfiles

Player data and profile system.

## Purpose
- Store persistent player data
- Profile UI display
- Stats tracking

## Data Tracked
- Level / XP
- Fame tier
- Playtime
- Join date
- Achievements (future)
- Bio/status message

## Features
- View own profile
- View other player profiles
- Edit bio/status
- Profile UI

## Dependencies
- HyCore
- HyFame (integration)
- HyEconomy (integration)
